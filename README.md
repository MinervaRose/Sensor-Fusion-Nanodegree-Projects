## Overview

This repository serves as an index for a collection of projects completed while studying autonomous systems, sensor fusion, computer vision, tracking, localization, planning, and control.

Together these projects cover a substantial portion of a modern autonomous vehicle perception and decision-making stack.

Topics include:

* Radar signal processing
* LiDAR obstacle detection
* Computer vision feature tracking
* Multi-object tracking
* Kalman filtering
* Sensor fusion
* Localization
* Path planning
* Vehicle control

## 🚧 LiDAR Obstacle Detection

3D point cloud processing pipeline using:

* RANSAC segmentation
* Euclidean clustering

* Bounding box generation
